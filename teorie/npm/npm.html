npm

npm, node package manager este o unealta care ne ajuta sa instalam / dezinstalam si importamn "packages" adica bucati de cod puse la dispozitie de catre altii pe npmjs.com

Cum se foloseste

Fiecare proiect va trebui sa aiba un "package.json" care va specifica ce dependinte avem. Aceste dependinte vor fi specificate in "dependencies" sau "devDependencies".

Instalarea pachetelor

Pentru a instala pachetele specificate in package.json rulam comanda "npm install".

Rularea comenzilor

Pentru a rula comenzile enumerate la rubrica "scripts" in package.json vom rula "npm run numelecomenzii". In cazul nostru avem npm run build-css

!!Gitignore!!

Atentie- Pentru a nu incarca pachetele in github va trebui sa cream un fisier numit .gitignore in care vom scrie "node_modules/". Acest gitignore va avea singurul rol de a spune gitului ce fisiere sau foldere sa ignore.
