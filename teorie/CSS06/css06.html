Bootstrap Library

Bootstrap functioneaza pe baza unui grid de 12 coloane, folosind .container si .rows putem avea coloanele respective specificate sub forma col-6 (adica jumatate din latime) sau col pentru latime automata.

Cel mai important lucru de retinut aici este ca trebuie sa folosim o structura bazata pe elemente cu clasa "container" care includ cate un element cu clasa "row" pt fiecare rand pe care vrem sa-l avem in container, si apoi in "row" avem elementele de tip "col" (sau variatii de genul "col-5")

Media Queries

Media queries, @media in CSS, sunt reguli speciale care spun CSS-ului pe ce criterii sa se aplice, sau nu.

Se numesc media queries pentru ca "intreaba" (to query) dipozitivul multimedia ce dimensiuni sau caracteristici are, si aplica anumite stiluri CSS pentru a suprascrie alt CSS.

  • min-width (specificat in pixeli)
  • max-width (specificat in pixeli)
  • orientation (portrait sau landscape)
