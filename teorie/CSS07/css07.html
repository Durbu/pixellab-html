Keyframes

Sintaxa

1. Declaratia keyframe-ului, si a setului de cadre (frames) cheie, incluzand numele, si cadrele denotate cu 0%. Aceste cadre pot avea oricate proprietati asociate. 
 @keyframes leftRight {
   0% {
      css-property: value;
   }
 }
2. Pe element trebuie sa punem keyframe-ul folosind regula CSS "animation". Animation poate primi un keyframe name, o durata, si daca sa ramana sau nu schimbat.
Pentru shorthand notation folosim in general timp scurt, cateva secunde si fie "infinite" fie "forwards" daca dorim sa controlam ce se intampla dupa rularea animatiei.

Min-height, max-height

Min & max width & height declara limitele unui element, astfel incat elementul nu va ocupa exact height-ul specificat cand ii dam 200px max-height ci va ocupa cat are el nevoie, pana in limita de 200px

Overflow

Proprietatea overflow primeste valori precum scroll, auto, hidden si se aplica in momentul in care elementele copii ies din limitele parintelui. Acest overflow poate fi de asemenea si specificat pe directie, si anume overflow-x si overflow-y. Overflow se aplica elementor copil cu "position:absolute" doar daca elementul cu overflow: hidden este cel putin "position:relative"
